## URL Structure Convention

```
/{section}/{action}/{id?}/{tab?}

Examples:
/agents                      ‚Üí List view
/agents/new                  ‚Üí Create new
/agents/abc123               ‚Üí Detail view
/agents/abc123/settings      ‚Üí Settings tab
/agents/abc123/analytics     ‚Üí Analytics tab
```
